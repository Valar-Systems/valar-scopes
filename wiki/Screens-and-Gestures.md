# Screens and Gestures

Blipscope is driven entirely by the touchscreen. There are three top-level screens, and a small indicator shows which one you're on.

## The three screens

### 📡 Radar
The live top-down radar view — see **[[Radar Display]]** for everything it shows.

### 📋 List
Every aircraft currently in range, as a scrollable list. Rows are colour-coded to match the radar: watchlisted contacts show in amber, and a pinned contact shows in white (the pin always wins). Tap a row to open that aircraft's **[[Aircraft Details|Aircraft-Details]]** card.

### 📊 Stats
At-a-glance statistics about the traffic Blipscope is currently seeing.

## Gestures

| Gesture | Where | What it does |
|---|---|---|
| **Tap** | Radar | Open the nearest aircraft's detail card |
| **Tap** | List | Open the tapped row's detail card |
| **Tap empty area** | Radar | Clear the pinned aircraft |
| **Swipe left** | Any screen | Go to the next screen |
| **Swipe right** | Any screen | Go to the previous screen |
| **Swipe up / down** | List | Scroll the list |
| **Tap** | Detail card | Flip photo page → data page, then close |
| **Swipe up** | Detail card | Pin ("track") this aircraft and return to the radar |
| **Any other swipe** | Detail card | Close the card |

Horizontal swipes cycle through the three screens in order, so you can flick left repeatedly to loop Radar → List → Stats → Radar.
